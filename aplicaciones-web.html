Aplicaciones Web

El usuario podrá acceder a los datos de modo interactivo, gracias a que la página responderá a cada una de sus acciones.

Este servicio permite al cliente pagar una cuota mensual o anual para usar la aplicación, sin necesidad de instalarla en la computadora del usuario.

Se pueden realizar tareas sencillas sin necesidad de descargar ni instalar ningún programa.
