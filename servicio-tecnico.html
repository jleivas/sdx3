Servicio técnico

Servicio técnico a empresas y privados, mantención y reparación de equipos computacionales.

Empresa orientada a gestión de servicios con experiencia en soporte tecnico, centrando foco de atención base a requerimientos que se generen del cliente.

  • Atención personalizada
  • Transparencia
  • Seguridad
  • Soporte remoto
  • Recuperación de datos
  • Optimización de sistemas operativos

Nuestros servicios

Formateo

El formateo de una unidad implica la eliminación de los datos, debido a que se cambia la asignación de archivos a sectores, con lo que se pierden los datos viejos y defectuosos que generan problemas de capacidad del equipo.

El computador quedará como nuevo, respaldando solo la información importante si el cliente lo requiere. Sistemas Operativos: Windows, Linux, Mac y Solaris.

Sistemas operativos

  • Windows 7, 8.1 y 10
  • Linux (todas las versiones)
  • Solaris

Mantenciones

Para Equipos de escritorio o notebooks, realizamos mantenciones preventivas mensualmente para tu pyme o empresa, garantizando la operatividad de los equipos, asistencia remota. planes y horarios a convenir con el cliente, consulte para más información.

Detalles del servicio

  • Toda la Región metropolitana
  • Solo soporte remoto para otras regiones
  • Estamos disponibles las 24 horas del día

Impresoras

Configuración de impresoras de su oficina. Instalación de drivers, conexiones en red local para su uso en todos los equipos.

Instalación de aplicaciones

Instalamos aplicaciones o software requeridas por el cliente, gratuitas o con licencias, también instalamos software desarrollado por softdirex.

Saber más acerca de nuestro servicio de desarrollo de software.

Reparación y mantención de Hardware

Reparamos desperfectos ocacionados por aspectos externos e internos del equipo y realizamos cambio de piezas y repuestos, además, asesoramos a nuestros clientes con consejos practicos para evitar el deterioro del hardware.

Soporte remoto

Solucionamos problemas de configuración de sistemas operativos de manera remota: Configuración y administración de sistema, Administración de redx, limpieza de disco, desfragmentación, eliminación de archivos dañinos, etc.

Configuración de red local

Instalamos y configuramos la red de área local para proveer a varios equipos y dispositivos de conexión tanto local como a internet ya sea mediante cableado o sistema inalambrico.

Respaldo o rescate de datos perdidos

Si el equipo dejó de funcionar y no te permite rescatar los datos mediante el sistema operativo, podemos recuperar total o parcialmente la información perdida y ofrecemos asesoría para respaldar la información importante de tu empresa. Puedes evitar estos problemas solicitando una Aplicación web para tu empresa.
